# 木材材數計算器 PWA — 部署教學

## 📁 檔案說明

```
timber-calculator-pwa/
├── index.html          ← 主程式（計算器）
├── manifest.json       ← PWA 設定檔（App 名稱、圖示）
├── sw.js               ← Service Worker（離線快取）
└── icons/
    ├── icon-192x192.png  ← App 圖示（小）
    └── icon-512x512.png  ← App 圖示（大）
```

---

## 🚀 部署到 GitHub Pages（免費）

### 第一步：建立 GitHub 帳號

1. 前往 https://github.com 註冊（免費）
2. 驗證 Email

### 第二步：建立 Repository

1. 登入後點右上角 **＋** → **New repository**
2. Repository name 填：`timber-calculator`
3. 選 **Public**
4. **不要**勾 Add a README
5. 點 **Create repository**

### 第三步：上傳檔案

**方法 A — 網頁直接上傳（最簡單）**

1. 在剛建立的 repository 頁面，點 **uploading an existing file**
2. 把解壓縮後的所有檔案拖進去（index.html、manifest.json、sw.js、icons 資料夾）
3. 在下方 Commit changes 填寫備註（例如「初次上傳」）
4. 點 **Commit changes**

> ⚠️ 注意：icons 資料夾需要先在 GitHub 上建立。
> 點 **Add file** → **Create new file**，檔名填 `icons/.gitkeep`，然後 commit。
> 接著再上傳 icon 圖片到 icons 資料夾。

**方法 B — 用 Git 命令（適合有經驗者）**

```bash
cd timber-calculator-pwa
git init
git add .
git commit -m "初次上傳"
git branch -M main
git remote add origin https://github.com/你的帳號/timber-calculator.git
git push -u origin main
```

### 第四步：開啟 GitHub Pages

1. 進入 repository → **Settings**（齒輪圖示）
2. 左側選單找到 **Pages**
3. Source 選 **Deploy from a branch**
4. Branch 選 **main**，資料夾選 **/ (root)**
5. 點 **Save**
6. 等 1～2 分鐘，頁面會顯示你的網址：

```
https://你的帳號.github.io/timber-calculator/
```

## 📱 安裝到手機 / iPad 主畫面

### iPhone / iPad（Safari）

1. 用 **Safari** 開啟上面的網址
2. 點底部的 **分享按鈕**（方框加向上箭頭 ⬆️）
3. 往下滑找到 **加入主畫面**
4. 確認名稱後點 **新增**
5. 主畫面就會出現「材數計算」的 App 圖示

### Android（Chrome）

1. 用 **Chrome** 開啟網址
2. 點右上角 **⋮** 選單
3. 選 **安裝應用程式** 或 **加到主畫面**
4. 確認後就會出現在桌面

### 電腦（Chrome / Edge）

1. 用 Chrome 開啟網址
2. 網址列右側會出現 **安裝** 圖示（⊕）
3. 點擊後確認安裝
4. 會像獨立 App 一樣開啟，沒有瀏覽器網址列

## 🔄 更新方式

要更新計算器內容時：

1. 到 GitHub repository 找到要修改的檔案
2. 點檔案 → 點鉛筆圖示（Edit）→ 修改 → Commit
3. 等 1～2 分鐘 GitHub Pages 會自動更新
4. **重要**：修改 `sw.js` 裡的版本號（`timber-calc-v1` → `timber-calc-v2`），這樣手機端才會更新快取

## 🏭 工廠現場使用

建議做法：
1. 部署完成後，用手機開啟網址
2. 把網址轉成 **QR Code**（搜尋「QR Code 產生器」即可）
3. 印出來貼在工廠牆上
4. 師傅用手機掃碼就能開啟，再加到主畫面就變成常駐 App
5. 安裝後即使沒有網路也能使用（離線功能）

## ❓ 常見問題

**Q：一定要用 GitHub 嗎？**
A：不一定。任何能託管靜態網頁的服務都行，例如 Netlify、Vercel、Cloudflare Pages，甚至自己的伺服器。GitHub Pages 是免費且最簡單的選擇。

**Q：需要付費嗎？**
A：GitHub Pages 對公開 repository 完全免費。

**Q：可以用自己的網域嗎？**
A：可以。在 GitHub Pages 設定裡有 Custom domain 選項，填入你的網域並設定 DNS 即可。

**Q：為什麼 iPhone 上沒有顯示安裝提示？**
A：iOS 不會主動跳安裝提示，需要手動透過 Safari 的分享按鈕 → 加入主畫面。注意必須用 Safari，Chrome/Firefox 在 iOS 上不支援 PWA 安裝。
